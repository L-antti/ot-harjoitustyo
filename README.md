### Asentaminen

1. **Kloonaa repositorio**:
   ```bash
   git clone git@github.com:L-antti/ot-harjoitustyo.git
   cd ot-harjoitustyo
   ```
2. **Asenna riippuvuudet**:
   ```bash
   poetry install
   ```
3. **Käynnistä sovellus**:
   ```bash
   poetry run invoke start
   ```
